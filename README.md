# Русские субтитры к сериалу Mortal Kombat: Conquest (1998–1999)

<img src="https://m.media-amazon.com/images/M/MV5BYmFhOTkxM2EtZGQ3Ny00Yzk5LWFlMzYtMjMzMDdmZmUzOGE1XkEyXkFqcGc@._V1_.jpg" width="400" alt="Mortal Kombat: Conquest">

В данном репозитории собраны отредактированные русские субтитры ко всем 22 сериям сериала [Mortal Kombat: Conquest](https://www.imdb.com/title/tt0170982/).

## Совместимость
Субтитры полностью синхронизированы с видеорядом. Названия файлов оставлены в оригинальном виде для корректного автоматического подцепления в современных медиаплеерах.

## История и авторы
Основой для проекта послужили субтитры, созданные и доработанные участниками сообщества **rutracker.org**. Оригинальное обсуждение проекта находится [в этой теме](https://rutracker.org/forum/viewtopic.php?t=2070941).

**Над оригинальным переводом и синхронизацией работали:**
*   [Kino-profan](https://rutracker.org/forum/profile.php?mode=viewprofile&u=15865191)
*   [lex2085](https://rutracker.org/forum/profile.php?mode=viewprofile&u=5564923)
*   [Belarus1983](https://rutracker.org/forum/profile.php?mode=viewprofile&u=51132140)

**Редакция (vdistortion):**
Тексты прошли полную сверку, а также комплексную орфографическую и литературную правку.

## Использование
Поместите файлы субтитров `.srt` в папку с соответствующими видеофайлами серий. Убедитесь, что их названия совпадают.
